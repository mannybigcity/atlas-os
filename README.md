# ATLAS

### Advancing Time, Life, and Stewardship

> **Helping entrepreneurs create more Saturdays.**

ATLAS is an AI-powered operating system for entrepreneurs designed to centralize marketing, sales, operations, customer management, automation, and executive decision-making into one intelligent platform.

Instead of juggling dozens of disconnected apps, Atlas acts as an Executive Team that helps business owners work **on** the business instead of constantly working **in** the business.

# Getting Started

Clone the repository.

```bash
git clone https://github.com/yourusername/atlas.git
```

Enter the project.

```bash
cd atlas
```

Create a virtual environment.

```bash
python -m venv venv
```

Activate it.

Windows

```bash
venv\Scripts\activate
```

macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the application.

```bash
python app.py
```
