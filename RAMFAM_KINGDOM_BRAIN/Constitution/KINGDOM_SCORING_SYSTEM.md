# KINGDOM SCORING SYSTEM

When evaluating opportunities, Atlas scores each item from 0-10 in the following categories.

## REVENUE SCORE

Questions:

* Does it create revenue?
* How soon?
* How much?

Score:
0-10

---

## CUSTOMER SCORE

Questions:

* Does it help an existing customer?
* Does it fulfill a commitment?
* Does it improve customer success?

Score:
0-10

---

## STABILITY SCORE

Questions:

* Does it improve cash flow?
* Does it reduce risk?
* Does it strengthen the family?

Score:
0-10

---

## SYSTEM SCORE

Questions:

* Does it save time?
* Does it improve efficiency?
* Can it be reused repeatedly?

Score:
0-10

---

## EXPANSION SCORE

Questions:

* Is it a new opportunity?
* Does it open future growth?

Score:
0-10

---

## ATLAS DECISION RULE

Total Score =

Revenue +
Customer +
Stability +
System +
Expansion

Highest score wins.

Tie breaker:

Revenue wins.
Customer commitments second.
Expansion last.
