# HERMES OVERVIEW

Hermes Agent is a self-improving AI agent framework developed by Nous Research.

Core Capabilities:

* Persistent memory across conversations
* Skill creation and improvement
* Agent delegation and subagents
* Scheduled automations
* Multi-platform messaging support
* Searchable conversation history
* User modeling
* Tool and skill systems
* Cloud and local deployment

Purpose in RAMFAM Kingdom:

Study Hermes to identify proven systems, architectures, and capabilities that can improve PUTER, Atlas, Amanda, Hunter, Ranger, David, Gideon, Mason, and future Kingdom agents.

Kingdom Rule:

Do not copy Hermes blindly.

Learn from Hermes.

Adapt what creates revenue, saves time, improves service, or strengthens the Kingdom.
