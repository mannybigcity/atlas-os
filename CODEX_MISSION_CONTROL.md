# Atlas Codex Mission Control

## First Law
Massive Action. Maximum Effort. Minimal Money.

## Codex Rules
1. Read existing files before creating new files.
2. Do not rebuild existing Atlas work without Manny approval.
3. Preserve dashboards, CRM, Gallery, client data, logos, colors, and assets.
4. Prefer small edits over rewrites.
5. Every task must report:
   - Files inspected
   - Files changed
   - Why this was minimal
   - How to test
6. Codex must use branches and pull requests, not direct main edits.

## Approval Gate
No merge unless Manny approves.
